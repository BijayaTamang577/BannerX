## Installation
+ Copy commands and paste it in Your Termux

```pkg update && pkg upgrade```
```pkg install git -y```
```git clone https://github.com/MrHacker-X/Termux-Shell.git```
```cd Termux-Shell```
```chmod +x *```
```bash setup.sh```
```exit```

## Single line command
```

```
